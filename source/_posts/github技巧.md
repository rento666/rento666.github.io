# 搜索技巧

## 1. in: 根据某个关键词来进行检索

关键词：

- **name**：按照项目名称搜索
- **description**：按照项目描述搜索
- **readme**：按照项目介绍搜索

语法：in:name 项目部分名字

## 2. stars 关注数

语法：stars:>1000

## 3. forks 分支数

语法：forks:>1000

## 4. language 语言

语法：language:Java

## 5. 项目更新时间

语法：pushed:>2020-01-01

## 6. 根据话题关联性

语法：topic:topic-name

> 每个条件之间用空格隔开即可。
